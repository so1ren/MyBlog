什么是变量？

变量是编程中最基本的概念之一。你可以把它想象成一个贴了标签的盒子， 盒子里面装着数据，标签上写着变量的名字。 关于这一点，廖雪峰老师的教程里有更详细的解释。

变量的命名规则

变量名必须遵循以下规则：

变量名必须以字母或下划线开头，后面跟任意数量的字母、数字或下划线。Python 区分大小写，所以 Name 和 name 是两个不同的变量。

来看一个具体例子：

python 
1name = "Alice"
2age = 25
3is_student = False
4print(f"{name} is {age} years old")

数据类型

Python 内置了丰富的数据类型。了解每种类型的特点是写出好代码的基础。 详细内容可以参考官方文档。

Python 是动态类型语言，这意味着你不需要显式声明变量的类型，Python 会自动推断。
